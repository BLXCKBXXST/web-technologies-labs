## Как снимать

- Экраны приложения — окно браузера, тёмная тема, без лишних вкладок.
- `04`, `13`–`15` — терминал с читаемым шрифтом.
- `10` — два окна браузера рядом: слева ведущий (видны элементы управления
  плеером), справа зритель (плеер без управления, та же позиция).
- `16` — адресная строка с замком и `https://blxckhub.server34.netcraze.club`.
