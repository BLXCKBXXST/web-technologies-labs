## Как снимать

- Экраны приложения — окно браузера в чистом виде, без лишних вкладок.
- `02`–`04` — экраны авторизации и профиля; `04` снимать, войдя как гость
  (видны плашка про удаление через 24 ч и бейдж «гость» в шапке).
- `12`–`15` — вывод команд (pytest, Vitest, `docker compose ps`, проверка
  HTTPS), оформлены как окно терминала.
