# Figma-каркасы blxck.hub

Самодостаточные HTML-каркасы экранов платформы **blxck.hub** для импорта в
Figma через плагин [html.to.design](https://www.figma.com/community/plugin/1159123024924461424).
Каждый файл — отдельная страница со встроенными стилями (один `<style>`,
без внешних ресурсов), поэтому импортируется независимо.

Вёрстка повторяет реализованную дизайн-систему приложения: тёмная тема,
индигово-фиолетовый акцент, токены из `app/frontend/src/styles/tokens.css`.
Видеопревью — заглушки-градиенты (в макете заменяются реальными кадрами).

## Состав

| Файл | Экран |
|------|-------|
| `home` | Главная — лента видео |
| `login` | Вход (пароль + гостевой доступ) |
| `register` | Регистрация |
| `watch` | Просмотр одиночного видео |
| `room` | Комната совместного просмотра (плеер + чат/Q&A) |
| `upload` | Загрузка видео |
| `profile` | Профиль пользователя |
| `notfound` | Страница 404 |

- `desktop/` — десктопная вёрстка, ширина кадра **1440 px**.
- `mobile/` — мобильная вёрстка, ширина кадра **390 px**.

Итого 16 файлов (8 экранов × 2 форм-фактора).

## Как импортировать в Figma

1. В Figma установить плагин **html.to.design**.
2. Plugins → html.to.design → вкладка **Import**.
3. Выбрать **Import from code/file** и загрузить нужный `.html`
   (или открыть файл, скопировать его содержимое и вставить).
4. Плагин создаст фрейм с разобранными слоями — каждый экран отдельным фреймом.

Импортировать файлы можно по одному; имена классов сохранены, поэтому слои
во Figma получают осмысленные названия.
