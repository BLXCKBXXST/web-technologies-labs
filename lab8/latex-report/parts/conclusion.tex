\chapter*{Заключение}
\addcontentsline{toc}{chapter}{Заключение}
\label{ch:conclusion}

В ходе выполнения лабораторной работы был развёрнут и настроен
веб-сервер на основе стека LAMP с системой управления контентом
WordPress в локальной сети учебного стенда.

В процессе работы были выполнены следующие задачи:
\begin{enumerate}
  \item Добавлены DNS-записи A и PTR для ВМ \texttt{wordpress}
        в BIND9 на сервере \texttt{gateway}.
  \item На новой ВМ \texttt{wordpress} установлен стек LAMP:
        Apache2, MariaDB, PHP~8.1 с необходимыми расширениями.
  \item Создана база данных \texttt{wordpress} и пользователь
        \texttt{author} с необходимыми привилегиями в MariaDB.
  \item Скачан, настроен и развёрнут WordPress в каталоге
        \texttt{/var/www/html}.
  \item Настроен Virtual Host Apache с поддержкой модуля
        \texttt{rewrite}.
  \item Завершена интерактивная установка WordPress через браузер
        с ВМ \texttt{desktop1}.
  \item Проверена работоспособность: создана и опубликована
        тестовая запись, подтверждён доступ к панели администратора.
\end{enumerate}

Все шаги автоматизированы bash-скриптами с идемпотентной логикой,
что позволяет повторно запускать их без риска нарушить уже
корректно настроенную систему.
