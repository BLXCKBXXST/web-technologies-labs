## 🖼️ Как делать скриншоты

- **Браузер (01–07)** — окно браузера целиком или скриншот вкладки. Для мобильного скриншота (06): F12 → Toggle device toolbar (Ctrl+Shift+M) → выбрать ширину 360 px.
- **Терминал (08)** — Linux: `gnome-screenshot -w`, `flameshot gui` или `Print Screen` с обрезкой; macOS: `Cmd + Shift + 4`, пробел, клик по окну; Windows: `Win + Shift + S`.

Файлы кладутся в `../img/` с именами **точно как в таблице** (`.png`, snake_case, без пробелов).

---

## ⚠️ Важно

- Имена файлов должны совпадать в трёх местах: эта таблица, `\includegraphics{...}` в `parts/chap3.tex`, и сами файлы в `../img/`.
- Скриншоты 01–06 можно снять на выдуманном персонаже из `config.js` — реальные данные для отчёта не нужны.
- Для скриншота 07 нужно дождаться, пока Caddy выпустит TLS-сертификат (~30 секунд после первого запроса по новому поддомену) — иначе в адресной строке будет ошибка вместо замка.
