## 🖼️ Как делать скриншоты

- **Терминал (01–04, 09)** — Linux: `gnome-screenshot -w`, `flameshot gui` или `Print Screen` с обрезкой; macOS: `Cmd + Shift + 4` затем пробел и клик по окну; Windows: `Win + Shift + S`.
- **Браузер (05–08)** — окно браузера целиком или скриншот вкладки. В DevTools для мобильного скриншота: F12 → Toggle device toolbar (Ctrl+Shift+M) → выбрать ширину 360 px → стандартный скриншот.

Файлы кладутся в `../img/` с именами **точно как в таблице** (`.png`, snake_case, без пробелов).

## ⚠️ Важно

- Имена файлов должны совпадать в трёх местах: эта таблица, `\includegraphics{...}` в `parts/chap3.tex`, и сами файлы в `../img/`.
- Для скриншота 08 нужно дождаться, пока Caddy выпустит TLS-сертификат (~30 секунд после первого запроса по новому поддомену) — иначе в адресной строке будет ошибка вместо замка.
- Для скриншота 05 учесть, что API на бесплатном Render-хостинге может холодно стартовать до 30 секунд — карточки появятся не сразу после открытия страницы.
