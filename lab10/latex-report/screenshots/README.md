## 🖼️ Как делать скриншоты

Два равнозначных способа:

1. **Export из Figma** (предпочтительно для самих макетов):
   - выделить фрейм (`Desktop 1440`, `Tablet 768`, `Mobile 360`);
   - правая панель → раздел `Export` → формат `PNG`, scale `1x`;
   - нажать `Export <название>`.

2. **PrintScreen окна браузера** (для скриншотов панелей, плагинов, сетки):
   - в Linux: `gnome-screenshot -w` или `flameshot gui`;
   - в Windows: `Win + Shift + S`;
   - в macOS: `Cmd + Shift + 4`, затем пробел → клик по окну.

Файлы клади в `../img/` с именами **точно как в таблице** (`.png`, без пробелов).
