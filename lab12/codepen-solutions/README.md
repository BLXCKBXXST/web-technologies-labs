## Порядок сдачи в Codepen

1. **Локально проверить решение.** Открыть нужный `index.html` в браузере,
   убедиться, что страница рендерится как ожидается (см. чек-лист ниже).
2. **Зайти на Codepen** под своим аккаунтом (или зарегистрироваться).
3. **Перейти на исходный pen** (ссылка `clck.ru/...` из таблицы выше).
4. **Нажать `Fork`** в правом верхнем углу — Codepen создаст копию pen-а
   на вашем аккаунте.
5. **Перенести изменения** из локального `index.html` (HTML-панель форка)
   и `styles.css` (CSS-панель форка). Внешние ссылки на стили MDL и
   Netology в Codepen уже подключены через настройки pen-а
   (`Settings → CSS → Add External Stylesheets`), их повторно вставлять
   не нужно.
6. **Нажать `Save`** — URL форка появится в адресной строке. Скопировать
   его и вставить в `latex-report/parts/chap3.tex` рядом с соответствующим
   блоком (плейсхолдер `TODO: URL форка`).

## Что должно получиться (быстрый чек-лист)

**Блок 1.1.** «Что такое нейросеть» — заголовок второго уровня (крупнее
обычного текста). Слово «научилась» во втором абзаце — кликабельная ссылка
(подчёркнута, цвет ссылки). Под заголовком «Источники и гейзеры Камчатки»
— три блока с подзаголовками и фотографиями (Малки, Налычевская долина,
Паратунка).

**Блок 1.2.** В первой секции — маркированный список из пяти имён. Во
второй — нумерованный список из пяти миллиардеров в порядке убывания
капитала (Гейтс → Безос). Третья таблица — 8 строк: 1 шапка + 7 строк
данных + ИТОГО на две ячейки + сумма 29 350 руб. в третьем столбце.
Четвёртая таблица — ровно 7 строк × 4 колонки, без «развалившихся» рядов.

**Блок 2.1.** Заголовок «Калуша с Калушатами» в `<div>` — синий.
Все упоминания Калушат (`<strong>`) — коричневые. Внутри цитат
`<blockquote>` упоминания Бутявки (`<span>`) — жёлтые жирные. В
нумерованном списке `<ol>` упоминания Калушат — курсивом.

**Блок 2.2.** Главный заголовок «Альберт Эйнштейн» — 56px, шрифт с
засечками, цвет `#365f91`. Все абзацы — 20px. В лид-абзаце слова
«Альбе́рт Эйнште́йн» — подчёркнуты. Список работ — с квадратами для
первого уровня и пустыми кружками для второго. Цитата — на фоновой
картинке, текст курсивом цвета `#494429`.
